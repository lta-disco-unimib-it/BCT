This folder contains the scripts necessary to create jars of the BCT applications.
The folder is organized in subfolders, one for each different application.
All the jar files are generated at the root of the workspace.
The jar export scripts are based on the assumption that the project containing BCT in eclipse is called BCT.

Here you can find a description of the folders and thei application:

bct 				contains the scripts to export the main bct.jat
bct-core-library	contains the scripts to create the core library used in the eclipse plugin it.unimib.disco.lta.bct.core
kbehavior 			scripts to export only the KBehavior jar


Requirement:
	the BCT project must be named BCT
